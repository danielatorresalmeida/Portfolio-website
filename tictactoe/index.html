Original C project

A modular terminal game built in C

The full C version includes configurable board sizes, custom win rules, save/load, undo, replay, computer difficulties, rankings, automatic tests, and 4-player or 8-player tournaments.

  • Multiple `.c` modules and a shared header
  • Colorblind-friendly terminal palette
  • Computer AI with easy, medium, and hard modes
  • Ranking, replay, save/load, and tournament output
